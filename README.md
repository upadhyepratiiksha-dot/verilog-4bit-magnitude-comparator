# 4-bit Magnitude Comparator

## Overview
This project implements a 4-bit Magnitude Comparator in Verilog HDL using a hierarchical design approach.

## Features
- Compares two 4-bit binary numbers
- Generates:
  - A > B
  - A < B
  - A = B

## Files

| File          | Description |
|---------------|-------------|
| mag_comp.v    | Top-level comparator module |
| adder.v       | Supporting module used in the design |
| mag_comp_tb.v | Testbench |

## Tools Used

- Verilog HDL
- Xilinx Vivado
- ModelSim
